
# REST API with TypeORM

This is a basic RESTful API using Node, Express, Typescript, and TypeORM.


## Installation

 ‎ 1. Clone repository
```bash
git clone https://github.com/Beirun/api_typeorm.git
```
‎‎‎  
‎ 2. Install packages

```bash
cd api_typeorm
npm install
```
‎‎  
‎ 3. Create **.env** file for Environment variables
```
api_typeorm   
└─src
└─node_modules
└─config.json
└─package-lock.json
└─package.json
└─tsconfig.json
└─.env
```
‎‎  
‎ 4. Run dev script
```bash
npm run dev
```
